# Audio assets

- En la **raíz de public**: **lofi.mp3** — Música de fondo ambiental (loop, volumen 0.12). Se reproduce al entrar a GameScreen; si el navegador bloquea autoplay, arranca en el primer clic/touch.
- **sfx/correct.mp3** — SFX match correcto.
- **sfx/incorrect.mp3** — SFX match incorrecto.
- **sfx/drag.mp3** — SFX al iniciar arrastre de flecha.
- **sfx/complete.mp3** — SFX al completar todas las conexiones.

Colocar los archivos aquí para que el juego reproduzca música y SFX.
